# loopAI

## 这是什么

一个基于 ReAct（Reasoning + Acting）范式的 AI Agent 框架。核心不是 agent 循环本身，而是围绕循环的工程化基础设施（harness）：工具抽象、上下文组织、错误恢复、执行边界。附带 Web 前端用于实时展示 agent 的思考链路、工具调用和状态变化。首个业务验证场景为磁盘空间诊断与清理。

## 核心价值

让 AI Agent 不仅"能跑"，而且可靠、可观测、可扩展——从 harness 设计的深度思考出发，构建值得信任的 agent 系统。

### 不在范围内

- 多 Agent 协作 — 当前聚焦单 Agent + harness，多 Agent 是后续方向
- 生产级部署 — 学习探索项目，不追求生产可用性
- 对接所有 LLM 提供商 — 先支持主流 API（OpenAI 兼容接口），后续按需扩展

## 上下文

- 学习驱动项目，通过实践深入理解 ReAct 范式和 agent harness 设计

- 参考任务还包括：CPU/内存占用排查、端口占用排查、偶发问题根因定位
