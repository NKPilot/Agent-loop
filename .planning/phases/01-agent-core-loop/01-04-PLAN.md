## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| EventBus -> JSONL Logger | Events written to disk. File permissions restrict access. |
| JSONL Logger -> filesystem | Session logs at logs/sessions/. Contains agent activity including tool output. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Information Disclosure | JSONLLogger | mitigate | File permissions 0o600 immediately after open(); logs/sessions/ directory 0o700 |
| T-04-02 | Tampering | JSONLLogger | accept | Append-only prevents modification within session; no log integrity verification in Phase 1 |
| T-04-03 | Denial of Service | CLIAgentRenderer | accept | Rich Live display is best-effort; bounded queue may drop events for slow terminals -- acceptable for CLI |
| T-04-04 | Information Disclosure | JSONLLogger._write() | mitigate | NEVER write api_key to log; AgentConfig uses SecretStr; LLMClient never includes it in events |
</threat_model>
