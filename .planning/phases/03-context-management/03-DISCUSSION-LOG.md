## 压缩策略

**问题:** 上下文接近 75% 窗口时怎么压缩？
**选择:** 滑动窗口+摘要
**说明:** 保留最近 N 轮完整对话，旧消息用 LLM 自动生成摘要替代。每次压缩需要一次额外 LLM 调用。
**备选:** 纯截断（简单但丢上下文）、混合策略（三管齐下）

## Token 计数

**问题:** Token 计数用什么方案？
**选择:** tiktoken 近似（cl100k_base）
**说明:** 跨模型误差 5% 以内，75% 阈值下有足够安全 margin。预留 provider-tokenizer 接口可扩展。
**备选:** 字符数估算（误差 ~20%）、混合方案

## 溢出文件

**问题:** 超长工具输出写入溢出文件后，LLM 如何引用？
**选择:** 文件名引用
**说明:** 上下文中保留 `[工具输出已保存至: xxx.txt (123KB)]`，LLM 可用 Bash 工具后续读取。
**备选:** 摘要后丢弃、暂不做溢出
