<!-- Key integration points between frontend and backend. -->

Frontend → Backend flow:
```
[Start Agent] button click
  → uiStore.setActiveSession(null)  // prepare for new session
  → startSession(prompt, maxSteps)   // POST /api/sessions/start → returns { session_id }
  → uiStore.setActiveSession(session_id)
  → useSessionEvents(session_id)     // creates SSE connection to /api/sessions/{id}/stream
  → SSE events flow into eventStore  // AgentTimeline renders steps in real-time
  → If confirmation_required event: uiStore.setPendingConfirmation(event) → ConfirmationDialog opens
  → User clicks Approve/Reject: confirmCommand(sessionId, confirmationId, approved) → POST /api/sessions/{id}/confirm
  → Backend PermissionGuard.respond() → agent continues
  → session_end event: React Query invalidates sessions list → SessionList refreshes
```

Production serving flow:
```
Browser → FastAPI (port 8000)
  → GET / → StaticFiles → frontend/dist/index.html
  → GET /assets/* → StaticFiles → frontend/dist/assets/*
  → GET /api/sessions → API router
  → GET /api/sessions/{id}/stream → SSE router
```
