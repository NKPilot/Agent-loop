在 FastAPI 后端上构建会话 REST API 和 Agent 控制端点。会话端点（list/detail/delete/export）读取 JSONL 日志文件，提供会话历史浏览数据源（OBS-05）。控制端点（start/confirm）允许从 Web 前端启动 agent 会话和响应危险命令确认（BIZ-02）。同时从 main.py 的 run_session() 中提取组件工厂函数，使 CLI 和 Web 路径共享同一套组件创建逻辑。

Output: 5 个 REST 端点（list/detail/delete/export/start/confirm）+ 组件工厂函数 + 集成测试。

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client→REST API | 不受信任的 HTTP 请求到达 `/api/sessions/*` 端点 |
| HTTP handler→filesystem | API 端点读取/删除 JSONL 文件 |
| HTTP handler→Agent runtime | POST /api/sessions/start 触发 agent 执行 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-05 | Tampering | routes/sessions.py:delete_session | mitigate | 路径遍历防护：session_id 从文件名解析后，使用 `Path(log_dir) / filename` 构造完整路径。通过 `.glob(f"*_{session_id}.jsonl")` 匹配而非直接拼接路径。 |
