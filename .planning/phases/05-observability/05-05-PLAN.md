<objective>
构建左右两个核心面板：SessionList（左侧 260px）和 AgentTimeline（中间 flex-1）。SessionList 从 REST API 加载会话列表，支持搜索、状态标记、活跃高亮和右键菜单操作。AgentTimeline 从 Zustand eventStore 消费实时 SSE 事件，渲染步骤时间线，支持 LLMToken 增量流式渲染、自动滚动和键盘导航。StepCard 组件负责单步骤的渲染逻辑。ConnectionStatus 显示 SSE 连接状态。

Purpose: 这两个面板是用户与 agent 交互的主要界面——会话选择和时间线观察。它们消费 Plan 04 建立的状态管理层（React Query + Zustand + useSSE hook）。
Output: 可交互的会话列表 + 实时 agent 时间线 + SSE 连接状态指示器。
</objective>
